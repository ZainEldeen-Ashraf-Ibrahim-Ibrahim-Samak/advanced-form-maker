# Quickstart: Admin Table Manager

## How to Test the Implementation

1. **Start the application**: `npm run dev`
2. **Access the Admin Form Builder**: Navigate to the form creation page.
3. **Add Table Component**: Drag and drop the "Table" component from the toolbox into the form.
4. **Configure Table**:
   - Click to configure the Table component.
   - Add columns (e.g., "Age", "Name").
   - Add pre-defined rows (e.g., "Person 1", "Person 2").
   - Toggle "Allow user to add rows" on or off.
5. **Preview Form**: Switch to the form preview and verify the table renders correctly.
6. **Translation Verification**: Run `npm run i18n:lint` to ensure all new UI keys are translated in both AR and EN.
